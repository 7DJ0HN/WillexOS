What is PositionSense?

PositionSense is a compact, BLE-powered sensor that mounts directly onto a gate leaf. It tracks real-world motion using both an accelerometer and a magnetic reed switch, giving true physical verification of the gate‚Äôs position ‚Äî not just what the motor thinks happened. Designed for installers, it works with any brand and requires no hinge wiring.

Technical Overview

  • ESP32 core with BLE communications
  • LIS3DH accelerometer for motion profiling
  • Reed switch + magnet for open/close alignment
  • IP65 compact enclosure
  • Deep-sleep battery optimisation
  • Humidity and temperature sensors as standard

Why PositionSense Matters

Traditional BUS systems and photocells only tell part of the story. PositionSense reads the leaf itself.

  • Eliminates blind spots in gate safety
  • Detects stuck, drifting, obstructed or wind-blown gates
  • Provides actionable early-warning fault insights
  • Useful for installers and homeowners alike

Works with All Gates

PositionSense works independently of motor brand logic. It does not rely on BUS, encoder feedback or internal motor assumptions. Instead, it measures the physical gate leaf directly.

  • Brand-agnostic ‚Äî works with any system
  • Can integrate seamlessly with WillexOS
  • Can also operate standalone for basic reporting

AI Diagnostics (with WillexOS)

When paired with a WillexOS controller, PositionSense contributes environmental, motion and behaviour data. Over time, this enables:

  • Predictive maintenance suggestions
  • Learning gate behaviour patterns
  • Weather-linked fault prediction
  • Deeper diagnostic insights for installers

Optional Standalone Mode

For sites without a WillexOS controller, PositionSense can operate via a dedicated BLE gateway module, optionally paired with a cloud dashboard for monitoring and alerts.
